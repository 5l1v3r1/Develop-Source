## Git操作指南

```
git status

git add .

git commit -m "注释"

git stash # 每次 push 前

git pull --rebase

# 如果有冲突，解决冲突
git rebase --continue

# gerrit review
git push origin HEAD:refs/for/master

# or

# github
git push origin develop 

git pull

git stash pop
```
